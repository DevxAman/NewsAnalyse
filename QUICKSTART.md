# 🚀 Quick Start Guide

## Run the Application

1. Open terminal in project directory
2. Run: `python app.py`
3. Open browser: `http://localhost:5000`
4. Upload PDF files and analyze!

## First Time Setup

If you haven't installed dependencies yet:
```bash
pip install -r requirements.txt
```

## Notes
- First run downloads ML models (~2GB) - be patient!
- Models load into memory (~4GB RAM needed)
- GPU automatically used if available
